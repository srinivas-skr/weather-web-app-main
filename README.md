

# Weather Web App

This is a simple static weather web application that displays weather data for any city entered by the user.

## Features
- Enter a city name to view its weather information
- Built using only HTML, CSS, and JavaScript
- No backend or server required

## How to Run
1. Open `home.html` in your web browser.
2. Navigate to other pages (`weather.html`, `about.html`) as needed.

## Project Structure
- `home.html`, `weather.html`, `about.html` - HTML pages
- Images and other assets are in the project root or subfolders

## Notes
- No installation or server setup is required.
- Works in any modern web browser.
